# Requirments: 
```
python (3.x is recommended)

python-tk/python3-tk

pip

virtualenv
```

# Installation: 
```
$ virtualenv -p python3 MobileFindr_env

$ cd MobileFindr_env

$ source bin/activate

$ git clone https://github.com/tigerlyb/MobileFindr.git 

$ cd MobileFindr

$ pip install -r requirement.txt -U
```
